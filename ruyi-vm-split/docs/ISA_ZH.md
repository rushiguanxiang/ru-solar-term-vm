# 如语言指令集手册 v0.2

## 编码格式

每条指令由 **卦字节(Hex Byte)** + **字段词(Field Word, 22bit)** 组成。

### 卦字节 (8bit)
| 位域 | 名称 | 说明 |
|------|------|------|
| [0] | 乄(Yi) | 0=隐周期(静读影子寄存器), 1=显周期(执行操作) |
| [6:1] | 爻(Yao) | 六爻编码，映射64卦 |

### 字段词 (22bit)
| 位域 | 名称 | 说明 |
|------|------|------|
| [19:18] | 涟漪等级 | 0-3级，与纳音冲突取最大值 |
| [17] | 环选择 | 0=周天内存, 1=甲子环内存 |
| [14:11] | 地支 | 寻址偏移 / IO端口号 |
| [10:6] | 纳音 | 四象限冲突检测索引(0-29) |
| [5:2] | 天干 | 源寄存器选择 |
| [1:0] | 甲子相位 | 保留 |

## 操作语义表

| 外卦 | 卦名 | 助记符 | 语义 | 副作用 |
|------|------|--------|------|--------|
| 000 | 坤 | STORE | mem[addr] ← dst | 写入触发纳音四象限检测 |
| 001 | 艮 | LOAD | dst ← mem[addr] | 读取受十二长生衰减修饰 |
| 010 | 坎 | ADD | dst ← reg[0] + src | 生数运算 |
| 011 | 巽 | SHL | dst ← reg[0] << (src & 7) | 左旋移位 |
| 100 | 震 | JMP | PC ← memVal & 0x7F | 跳转并触发节气推进 |
| 101 | 离 | AND | dst ← reg[0] & src | 与门运算 |
| 110 | 兑 | IO | dizhi=6:输出 / dizhi=7:阻塞输入 | 外设交互 |
| 111 | 乾 | CTRL | src=0x55时开启羲和模式 | 特权控制 |
